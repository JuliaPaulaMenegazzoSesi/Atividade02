Tecnologias

Linguagem de Marcação de HiperTexto, ou apenas HTML, na criação de websites é o que estrutura a página. Nela, utilizam-se tags para definir os elementos da página. <p>, por exemplo, é utilizada para criação de parágrafos. A linguagem de marcação foi criada em 1993 por Tim Berners-Lee a fim de resolver problemas de comunicação com sua equipe, e atualmente se encontra na versão HTML5.

Cascading Style Sheets, ou apenas CSS, é o que modifica o desing de uma página. Nele, é possível modificar elementos como cores, fontes, posicionamento de elementos, imagens, entre outros. O suporte do CSS pode mudar de navegador para navegador, por isso é importante avaliar se certos elementos são compatíveis em certos navegadores. CSS foi desenvolvido em 1996 por Håkon Wium Lie e Bert Bos, e atualmente se encontra na versão CSS3.

JavaScript é o que torna possível a interatividade de uma página web, e atualmente é a tecnologia mais utilizada para isso na internet. A linguagem é suportada por todos os navegadores, e possui suporte de elementos da linguagem C, além de possuir tipagem dinâmica. A linguagem foi lançada em 1995, desenvolvida por Brendan Eich, e foi influenciada por diversas outras linguagens, tais como Python, C, e Java, com a qual é comumente confundida.
